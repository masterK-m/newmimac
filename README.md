# newmimac


웹에 관한 모든 것 스터디 오픈채팅방에 오신 것을 환영합니다!

이 스터디를 통해 
Node.js, javascript, 타입스크립트, 앵귤러.js, 알고리즘, 자료구조, 디자인패턴 등
흐지부지 되지만 않는다면 
점점 확장하여 공부를 하고자 합니다. 

설립 목적은 자기계발입니다.
회사를 위한 공부가 아닌, 나 자신을 위한 공부입니다.

커리큘럼을 정해놓고 
이론은 각자 공부 하고 실습 위주로 할 예정이며

깃허브를 통해 소스를 관리하고
소스를 공유하며 모임 때 
소스를 보며 코드 리뷰 방식으로 진행 할 예정입니다. 

스터디 후에는 
스터디 내용을 정리하여
깃허브나 오픈톡방에 공유할 예정입니다. 

2명만 모여도 스터디를 진행 할 예정이며
모든 의사결정은 
오픈 톡방의 투표에서 다수결로 진행 할 예정입니다. 

직급과 관계 없이 
질문을 많이 하는 스터디 방이 되었으면 하는 취지에서 
오픈 채팅방에서는 직급을 붙이지 않고
닉네임으로 불렀으면 합니다. 

첫 모임 때는 
깃허브에 대해 숙지하고 
깃허브 사용법에 관한 리뷰를 하려고 합니다.
(요즘 채용공고에 형상관리 Programs(SVN, Git) 경험자 우대한다는 내용이 많습니다.)

방에 처음 들어오신 분들께서는 
1. 닉네임 변경
2. 투표 참여 
해주시길 바랍니다.

스터디 주제가 바뀔 때 마다 
방을 나가셔도 들어오셔도 됩니다. (참여코드 : mimac)

실습과 과제는 의무가 아닙니다.
실습과 과제를 하지 않아도 
코드리뷰에 참석하실 수 있으며 
스터디 내용에 이해하지 못하고 지나치는 사람 없이 진행이 될 수 있도록 
서로 노력하였으면 좋겠습니다. ^^*
